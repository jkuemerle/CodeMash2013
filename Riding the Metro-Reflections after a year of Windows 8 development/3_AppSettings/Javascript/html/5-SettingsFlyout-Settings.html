Basic controls for settings

The following controls are a good starting list of controls to build your settings UI.

Toggle switch

Use toggle switches to let users set Boolean values.


Push button

With a push button, users initiate an immediate action.

Select control

Use the select to allow users to select one item from a set of text only items.

Hyperlink

Use a hyperlink when the associated action will take the user out of this flyout.
