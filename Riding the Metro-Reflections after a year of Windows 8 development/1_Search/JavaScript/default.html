Description:

Participating in the Search Extension lets users search with your app from anywhere in the system (including another app) by selecting your app in the Search charm.

Follow these steps to see how users select an app (with this extension) to search from the Search charm:

1. Open charms bar
2. Select Search
3. Type in a query
4. Submit the query
5. Select a different app in the search pane
6. Submit a new query
7. Switch back to Search Extension Sample app in the Search Pane
